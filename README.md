# File Encryption Tool (C++)

## What it does  
This is a simple C++ program that encrypts and decrypts files using a key. It uses XOR encryption, so the same key is used for both encrypting and decrypting.

## How to run  

Compile:
```
g++ encrypt.cpp -o encrypt
```

Run:
```
./encrypt
```

## How to use  
1. Choose encrypt or decrypt  
2. Enter input file name  
3. Enter output file name  
4. Enter your key  

## Note  
This is a basic project for learning purposes and not meant for real-world security.
